# MeshComm Quick Start Guide

## For EMS Main Computer:
1. Double-click `start_main.bat`
2. Open `http://localhost:3001` in your browser

## For Client Computers:
1. Copy this entire folder to the client computer
2. Double-click `start_client.bat`
3. Open `http://localhost:3002` in your browser

## Notes:
- Make sure Go is installed on all computers
- Both computers must be on the same network
- The client will automatically discover and connect to the main node
- Messages will sync between all connected computers
